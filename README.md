# 玩Android-Flutter客户端

## 前言

- 这是一款使用Kotlin编写的，MaterialDesign风格的WanAndroid客户端应用
- 这是一个可以用来入门Kotlin的项目
- [项目完全开源](https://github.com/shijiacheng/wanandroid_kotlin)，欢迎Star&Fork，有问题请提交Issues，个人项目，请勿商用
